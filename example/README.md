# Examples

This directory contains examples of how to use the `dict-trans-tokenizer` package.

## Running the Examples

Before running the examples, make sure you have:

1. Installed fast_align (required for token alignment)

> [!NOTE]
> Use the forked fast_align (<https://github.com/FremyCompany/fast_align>).

### Basic Usage

```bash
uv run example/basic_usage.py
```
